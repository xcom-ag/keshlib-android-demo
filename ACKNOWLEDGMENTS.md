## Summary
The keshlib for Android uses code from the following libraries

* socket.io by Enno Boland https://github.com/Gottox/socket.io-java-client/
* java_websocket by Nathan Rajlich https://github.com/TooTallNate/Java-WebSocket/
* JSON by JSON.org http://www.json.org/
